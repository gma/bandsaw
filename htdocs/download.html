Band Saw has not yet been released, but the latest version is available from CVS. Once a useful version is available a package will be made available here.

Requirements

  • Python (2.3 or later)
  • GTK+ (2.0 or later)
  • PyGTK

The client should run on any operating system that supports Python, GTK/PyGTK and FIFOs (named pipes).
