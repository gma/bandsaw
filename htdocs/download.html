Download

Band Saw is now available for download:

  • bandsaw-0.1.0.tar.gz
  • bandsaw-0.1.0.tar.bz2

Alternatively, the latest code is always available from CVS.

Requirements

  • Python (2.3 or later)
  • GTK+ (2.0 or later)
  • PyGTK and gnome-python

It should run on any operating system that supports GNOME, Python and named pipes (otherwise known as FIFOs).

Installation

Installation is simple, but please read the README file included with the source code. It is important that you specify the correct prefix when configuring the package or you may find that some things don't work correctly on your system (e.g. default configuration parameters, the user manual, etc.).

Update: In the README I wrote that you should run make check before installing, but I forgot that most people won't have the pMock Python package installed, which will cause make check to fail. Either skip make check or install pMock. Band Saw will work without any problems even if you can't run the tests; pMock is only required during development. I'll be removing make check from the README and releasing version 0.1.1 just as soon as I feel I've left enough time for feedback about other similarly niggly little problems.
